# Password Manager Application

A project that aims to ease password, note ,card and address information storage without compromising on security and relying solely on memory which on occation may sometimes be unreliable.
This project aims to solve that by providing an easy to use, secure platform that eables you to store all you sensitive information like passwords all on a unified single master password.
The application focuses on security, usability, and efficient data organization.

⚙️ How It Works

The application requires users to create or enter a master password to gain access. Once authenticated, users can store and manage different types of sensitive data. All data is securely stored and retrieved from a database, with encryption applied to protect critical information.
